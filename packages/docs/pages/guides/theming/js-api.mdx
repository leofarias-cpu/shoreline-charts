# JS API

Instead of using Shoreline's CLI the theme files generation can be made programmatically using only `@vtex/shoreline-css`. Consider the project with the structure:

<FileTree>
  <FileTree.Folder name="project" defaultOpen>
    <FileTree.File name="shoreline.config.js" />
    <FileTree.File name="index.js" />
    <FileTree.File name="package.json" />
  </FileTree.Folder>
</FileTree>

The configuration file is:

```js filename="shoreline.config.js"
import { defineConfig } from '@vtex/shoreline-css'

export default defineConfig({
  preset: 'base',
  tokens: {
    color: {
      primary: 'oklch(54% 0.212 340.41)',
    },
  },
})
```

The main script should call the async function `theme`, that has the return type `Promise<void>`.

```js filename="index.js"
import { css } from '@vtex/shoreline-css'

async function generate() {
  await css()
}

generate()
```

The script runs on a node environment.

```json filename="package.json"
{
  "scripts": {
    "start": "node index.js"
  }
}
```

The call to `theme()` generates the contents of the `shoreline` folder.
