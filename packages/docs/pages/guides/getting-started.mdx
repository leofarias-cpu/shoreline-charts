# Getting Started

Install Shoreline and start building your project in minutes.

## Installation

Getting up and running is quick and easy.

<Steps>

### Create a new project

If you are starting a new project you can refer to the Raccoon section of the documentation to create a new project.

<Cards>

<Card title="Start with Raccoon" href="../raccoon/getting-started" />

</Cards>

### Install Shoreline components

```sh npm2yarn copy
npm install @vtex/shoreline-components
```

### Install Sunrise theme

```sh npm2yarn copy
npm install @vtex/shoreline-theme-sunrise
```

### Import the CSS theme file

Import the theme file at the root of your application.

```jsx copy
import '@vtex/shoreline-theme-sunrise/css'
```

### Start building

You're all set! Start building your project with Shoreline.

```jsx
import { Button } from '@vtex/shoreline-components'

export default function Home() {
  return (
    <Button>Let's build!</Button>
  )
}
```

</Steps>
