## Top-level folders

A Raccoon project mixes [VTEX-IO builders](https://developers.vtex.com/docs/guides/vtex-io-documentation-builders) with [NextJS](https://nextjs.org/) files.

| folder   | description                                                                                                                                                                                                         |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| admin    | The admin [builder](https://developers.vtex.com/docs/guides/vtex-io-documentation-builders). Docs on how to configure the [navigation.json](https://tech-docs.vtex.com/Admin/admin-shell/#admin-apps-configuration) |
| messages | The [messages builder](https://tech-docs.vtex.com/Admin/admin-shell/#the-messages-folder)                                                                                                                           |
| pages    | [NextJS pages](https://nextjs.org/docs/pages/building-your-application/routing)                                                                                                                                     |
| public   | [NextJS static assets](https://nextjs.org/docs/app/building-your-application/optimizing/static-assets)                                                                                                              |
