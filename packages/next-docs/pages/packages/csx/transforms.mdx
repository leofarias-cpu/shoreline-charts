# Transforms

## bg

Transforms a the `bg` key into `background`.

```js
const style = csx({
  bg: 'black',
})
```

Becomes:

```js
const style = {
  background: 'black',
}
```

## fg

Transforms a the `fg` key into `color`. Fg stands for foreground.

```js
const style = csx({
  fg: 'black',
})
```

Becomes:

```js
const style = {
  color: 'black',
}
```

## margin

Transforms the `margin` tokens deeply.

```js
const style = csx({
  margin: '1rem $space-1 1rem',
})
```

Becomes:

```js
const style = {
  margin: '1rem var(--sl-space-1) 1rem',
}
```
