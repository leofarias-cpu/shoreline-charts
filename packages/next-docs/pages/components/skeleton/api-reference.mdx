# API Reference

Skeleton represents a loading state

```jsx
<Skeleton shape="circle" style={{ width: 200, height: 200 }} />
```

## Parameters

### `props`

- Type: `@vtex/shoreline-components.SkeletonProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `shape` (optional)

Skeleton shape

- Type: `circle | rect`

- Default: `rect`
