# API Reference

Confirmation Modal containers allow merchants to confirm an action through an overlay window that opens on top of the current page.

## Parameters

### `props`

- Type: `@vtex/shoreline-components.ConfirmationModalProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `locale` (optional)

The BCP47 language code for the locale.

- Type: `string`

- Default: `en-US`

### `messages` (optional)

Object containing all messages to be displayed internally in the modal. They include "title", "ok" and "cancel".

- Type: `{ cancel: string; confirm: string; title: string; }`

### `onCancel` (optional)

Callback fired when the cancel button is clicked.

- Type: `@types/react.React.MouseEventHandler<typescript.HTMLButtonElement>`

### `onClose` (optional)

- Type: `{ (event: typescript.Event): void; }`

### `onConfirm` (optional)

Callback fired when the ok button is clicked.

- Type: `@types/react.React.MouseEventHandler<typescript.HTMLButtonElement>`

### `open` (optional)

Whether the modal is open or not.

- Type: `boolean`

- Default: `false`
