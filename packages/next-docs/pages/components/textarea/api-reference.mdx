# API Reference

Textarea fields allow merchants to enter multiple lines of text.

```jsx
<Textarea label="Label" maxLength={120} optional resizable={false} />
```

## Parameters

### `props`

- Type: `@vtex/shoreline-components.TextareaProps & @types/react.React.RefAttributes<typescript.HTMLTextAreaElement>`

## Props

### `error` (optional)

Whether is in error state

- Type: `boolean`

### `onChange` (optional)

Callback for value change

- Type: `@types/react.React.Dispatch<any>`

### `optional` (optional)

Whether the textarea is optional or not

- Type: `boolean`

### `resizable` (optional)

Whether the textarea is resizable or not

- Type: `boolean`

- Default: `true`
