# API Reference

Loading indicator

```jsx
<Spinner description="loading" size={16} />
```

## Parameters

### `props`

- Type: `@vtex/shoreline-components.SpinnerProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `description` (optional)

Specify a description that would be used to best describe the loading state

- Type: `string`

- Default: `'loading'`

### `size` (optional)

Increase or decrease the size of the shape

- Type: `number`

- Default: `16`
