# Spinner

Loading indicator

```jsx copy showLineNumbers
<Spinner description="loading" size={16} />
```

## Parameters

| Name    | Type                                                                                                                                              |
| :------ | :------------------------------------------------------------------------------------------------------------------------------------------------ |
| `props` | [`SpinnerProps`](/components/spinner/props.md) & `RefAttributes`\<[`HTMLDivElement`](https://developer.mozilla.org/docs/Web/API/HTMLDivElement)\> |

## Props

### description

• `Optional` **description**: `string`

Specify a description that would be used to best describe the loading state

**`Default`**

```ts
"loading";
```

---

### size

• `Optional` **size**: `number`

Increase or decrease the size of the shape

**`Default`**

```ts
16;
```
