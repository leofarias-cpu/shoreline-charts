# Props

## Optional Props

### `Common Props (div)`

All built-in browser components, such as `<div>`, support some [common props and events.](https://react.dev/reference/react-dom/components/common)

### `size`

- Type: `number`
- Default: `24`

Increase or decrease the size of the shape.

### `description`

- Type: `string`
- Default: `'loading'`

Specify a description that would be used to best describe the loading state
