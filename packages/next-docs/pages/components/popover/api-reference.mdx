# API Reference

Popover containers allow merchants to access and interact with more
information of an item through an overlay.

```jsx
<PopoverProvider>
  <PopoverTrigger>Trigger</PopoverTrigger>
  <Popover>Content</Popover>
</PopoverProvider>
```

## Parameters

### `props`

- Type: `@vtex/shoreline-components.PopoverProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `asChild` (optional)

Children composition

- Type: `boolean`

- Default: `false`
