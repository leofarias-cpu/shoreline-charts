# Pagination

Pagination triggers allow merchants to view the size of a list and navigate between pages.

```jsx copy showLineNumbers
<Pagination page={page} onPageChange={(page, type) => {}} total={total} />
```

## Parameters

| Name    | Type                                                                                                                                                    |
| :------ | :------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `props` | [`PaginationProps`](/components/pagination/props.md) & `RefAttributes`\<[`HTMLDivElement`](https://developer.mozilla.org/docs/Web/API/HTMLDivElement)\> |

## Props

### loading

• `Optional` **loading**: `boolean`

Whether the pagination is loading or not.

**`Default`**

```ts
false;
```

---

### onPageChange

• `Optional` **onPageChange**: (`page`: `number`, `type`: `"next"` \| `"prev"`) => `void`

#### Type declaration

▸ (`page`, `type`): `void`

Function called whenever the pagination actions are triggered and the page changes.

##### Parameters

| Name   | Type                 | Description                               |
| :----- | :------------------- | :---------------------------------------- |
| `page` | `number`             | The new page number reference             |
| `type` | `"next"` \| `"prev"` | The type of the action that was triggered |

##### Returns

`void`

---

### page

• **page**: `number`

Pagination current page

---

### size

• `Optional` **size**: `number`

Page size

**`Default`**

```ts
25;
```

---

### total

• **total**: `number`

The total number of items
