# VisuallyHidden

Renders an element that is visually only to screen readers

#### Parameters

| Name    | Type                                                                                                                                                             |
| :------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `props` | [`VisuallyHiddenProps`](/components/visually-hidden/props.md) & `RefAttributes`\<[`HTMLDivElement`](https://developer.mozilla.org/docs/Web/API/HTMLDivElement)\> |

**`Example`**

```jsx copy showLineNumbers
<a href="#">
  Learn more<VisuallyHidden> about brazilian beaches</VisuallyHidden>
</a>
```
