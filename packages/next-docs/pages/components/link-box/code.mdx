# Code

```jsx
<LinkBox href="">
  <div>
    <a>...</a>
  </div>
  <button>...</button>
</LinkBox>
```

## Parameters

### `props`

- Type: `@vtex/shoreline-components.LinkBoxProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `asChild` (optional)

Enable children composition

- Type: `boolean`

- Default: `false`

### `href`

The URL that the hyperlink points to.

- Type: `string`

### `target` (optional)

Where to display the linked URL

- Type: `@vtex/shoreline-components.NavigationTarget`

- Default: `'_parent'`
