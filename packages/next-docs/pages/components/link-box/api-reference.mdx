# API Reference

A container that acts as a link. It allows text selection and stop its children event propagation.

```jsx
<LinkBox href="">
  <div>
    <a>...</a>
  </div>
  <button>...</button>
</LinkBox>
```

## Parameters

### `props`

- Type: `@vtex/shoreline-primitives.LinkBoxProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `asChild` (optional)

Enable children composition

- Type: `boolean`

- Default: `false`

### `href`

The URL that the hyperlink points to.

- Type: `string`

### `target` (optional)

Where to display the linked URL

- Type: `@vtex/shoreline-primitives.NavigationTarget`

- Default: `'_parent'`
