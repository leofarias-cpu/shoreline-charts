# API Reference

Represents clickable surfaces

```jsx
<Clickable>
  Text <button>Click Me</button>
</Clickable>
```

## Parameters

### `props`

- Type: `@vtex/shoreline-primitives.ClickableProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `asChild` (optional)

Enable children composition

- Type: `boolean`

- Default: `false`
