# API Reference

Collections containers allow merchants to view and control a set of items in one or more layout types.

```jsx
<Collections>
  <Slot name="header">
    <Slot name="controls">
      <Search />
      <Pagination page={1} total={74} />
    </Slot>
  </Slot>
  <CollectionView status="ready">
    <div className="ready-view" />
  </CollectionView>
  <Slot name="footer">
    <Pagination page={1} total={74} />
  </Slot>
</Collections>
```

## Parameters

### `props`

- Type: `typescript.Omit<@types/react.React.DetailedHTMLProps<@types/react.React.HTMLAttributes<typescript.HTMLDivElement>, typescript.HTMLDivElement>, ref> & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `CollectionsProps`

- Type: `@types/react.React.ComponentPropsWithoutRef<div>`
