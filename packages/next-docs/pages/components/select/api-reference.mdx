# API Reference

Select fields allow merchants to choose a single option from a list
that includes between five and seven values.

```jsx
<Select label="Label" helpText="Help text" errorText="Error text">
  <option>option</option>
  <option>option</option>
  <option>option</option>
</Select>
```

## Parameters

### `props`

- Type: `@vtex/shoreline-components.SelectProps & @types/react.React.RefAttributes<typescript.HTMLSelectElement>`

## Props

### `disabled` (optional)

Whether is disabled

- Type: `boolean`

- Default: `false`

### `error` (optional)

Whether is in error state

- Type: `boolean`

### `onChange` (optional)

Callback for value change

- Type: `@types/react.React.Dispatch<any>`
