# API Reference

```jsx
<Container>

</Container>
```

### `props`

- Type: `@vtex/shoreline-components.ContainerProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `asChild` (optional)

- Type: `boolean`

- Default: `false`
