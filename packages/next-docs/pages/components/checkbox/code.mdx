# Checkbox

Checkbox controls allow users to make multiple independent choices in a form where there are at most five options and each option is binary.

```jsx copy showLineNumbers
<Checkbox>Label</Checkbox>
```

## Parameters

| Name    | Type                                                                                                                   |
| :------ | :--------------------------------------------------------------------------------------------------------------------- |
| `props` | `AriaCheckboxProps` & `RefAttributes`\<[`HTMLDivElement`](https://developer.mozilla.org/docs/Web/API/HTMLDivElement)\> |
