# API Reference

Spaces elements consistently

```jsx
<Stack>
  <div>Stacked 1</div>
  <div>Stacked 2</div>
</Stack>
```

## Parameters

### `props`

- Type: `@vtex/shoreline-components.StackProps & @types/react.React.RefAttributes<typescript.HTMLDivElement>`

## Props

### `align` (optional)

items alignment

- Type: `end | start`

- Default: `start`

### `asChild` (optional)

Children composition

- Type: `boolean`

- Default: `false`

### `fluid` (optional)

Grows the width of items to match the parent

- Type: `boolean`

- Default: `false`

### `horizontal` (optional)

Switches the layout to horizontal

- Type: `boolean`

- Default: `false`

### `space` (optional)

space between items

- Type: `string`

- Default: `0`
