# API Reference

An alert is an element that displays a brief, important message in a way that attracts the user's attention without interrupting the user's task.

```tsx
<Alert>Alert description</Alert>
```

## Optional Props

### `onDismiss`

Action that is called once the alert is dismissed.

- Type: `() => void`
- Default: `undefined`

### `variant`

The tone of voice.

- Type: `'info' | 'positive' | 'warning' | 'critical'`
- Default: `'info'`

### `children`

Component children.

- Type: `React.ReactNode`
- Default: `undefined`

### `action`

An action that a user can take upon the alert.

- Type: `React.ComponentProps<Button>`
- Default: `undefined`
