# Namespaces

Namespaces are a collection of css rules manipulated in a specific maner.

## @media

Reserved namespace for media queries [Media Queries](https://developer.mozilla.org/en-US/docs/Web/CSS/@media).

<Tabs items={['input', 'output']}>

<Tab>

```ts
const styles = {
  '@media': {
    tablet: {
      size: 100,
    },
    darkMode: {
      bg: 'black',
    },
  },
}
```

</Tab>

<Tab>

```ts
const styles = {
  '@media': {
    '(min-width: var(--sl-bp-tablet))': {
      width: 100,
      height: 100,
    },
    '(prefers-color-scheme: dark)': {
      background: 'black',
    },
  },
}
```

</Tab>

</Tabs>

### Core medias

Available core auto-completions:

| property     | transform result                       |
| ------------ | -------------------------------------- |
| `tablet`     | `(min-width: var(--sl-bp-tablet))`     |
| `desktop`    | `(min-width: var(--sl-bp-desktop))`    |
| `widescreen` | `(min-width: var(--sl-bp-widescreen))` |
| `darkMode`   | `(prefers-color-scheme: dark)`         |

## @layer

Represents a [Cascade Layer](https://developer.mozilla.org/en-US/docs/Web/CSS/@layer).

<Tabs items={['input', 'output']}>

<Tab>

```ts
const styles = {
  '@layer': {
    reset: {
      '*': {
        boxSizing: 'border-box'
      }
    },
    tokens: {
      ':root' {
        '--sl-bg-primary': 'pink'
      }
    }
  },
}
```

</Tab>

<Tab>

```ts
const styles = {
  '@layer': {
    'sl-reset': {
      '*': {
        boxSizing: 'border-box'
      }
    },
    'sl-tokens': {
      ':root' {
        '--sl-bg-primary': 'pink'
      }
    }
  },
}
```

</Tab>

</Tabs>

### Core layers

Shoreline has 4 default layers:

| property     | transform result |
| ------------ | ---------------- |
| `reset`      | `sl-reset`       |
| `base`       | `sl-base`        |
| `tokens`     | `sl-tokens`      |
| `components` | `sl-components`  |

## Supports

The [@supports](https://developer.mozilla.org/en-US/docs/Web/CSS/@supports) at-rule lets you specify CSS declarations that depend on a browser's support for CSS features.

```ts
const style = csx({
  '@supports': {
    '(background: color(display-p3 0 1 0))': {
      bg: 'color(display-p3 0 1 0)',
    },
  },
})
```
