## Accessibility

<OrderedList>
  <OrderedListItem>`Tooltip` has `role="tooltip"`.</OrderedListItem>
  <OrderedListItem>
    Tooltip's children has `aria-describedby` referring to the `Tooltip`.
  </OrderedListItem>
  <OrderedListItem>
    <kbd>Esc</kbd> hides the current visible tooltip.
  </OrderedListItem>
