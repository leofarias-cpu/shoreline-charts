<IntroSection>

## What is this?

Columns is a layout component that makes it easy to implement horizontally layouts while defining widths and space between them.

</IntroSection>

<Section hasExplanation>

## Auto

Columns fits the space with equal widths when the quantity of `units` aren't specified.

```jsx live
<Columns>
  <Column>
    <Box csx={{ ...palette('purple'), padding: 2 }}>6 units</Box>
  </Column>
  <Column>
    <Box csx={{ ...palette('teal'), padding: 2 }}>6 units</Box>
  </Column>
  <Column>
    <Box csx={{ ...palette('purple'), padding: 2 }}>6 units</Box>
  </Column>
</Columns>
```

</Section>

## Units

All columns are of equal width by default, but you can also customise the width of each column individually.

```jsx live
<Columns spacing={1}>
  <Column units={3}>
    <Box csx={{ ...palette('purple'), padding: 2 }}>3 units</Box>
  </Column>
  <Column units={6}>
    <Box csx={{ ...palette('teal'), padding: 2 }}>6 units</Box>
  </Column>
  <Column units={3}>
    <Box csx={{ ...palette('purple'), padding: 2 }}>3 units</Box>
  </Column>
</Columns>
```

</Section>

<Section hasExplanation>

## Offset

It is possible to define offsets between columns.

```jsx live
<Columns space="0">
  <Column units={3}>
    <Box csx={{ ...palette('purple'), padding: 2 }}>3 units</Box>
  </Column>
  <Column offset="both" units={3}>
    <Box csx={{ ...palette('teal'), padding: 2 }}>3 units</Box>
  </Column>
  <Column units={3}>
    <Box csx={{ ...palette('purple'), padding: 2 }}>3 units</Box>
  </Column>
</Columns>
```

</Section>

<Section hasExplanation>

## Responsive Prop

To make it easy applying columns layouts, all props provided by `Columns` are [Responsive Props](/next/introduction/developing/styling#responsive-prop).

```jsx live
<Columns space={{ mobile: '0', tablet: '$m' }}>
  <Column
    units={{ mobile: 12, tablet: 6 }}
    offset={{ mobile: 'none', tablet: 'right' }}
  >
    <Box csx={{ ...palette('purple'), padding: 2 }}>6 units</Box>
  </Column>
  <Column units={{ mobile: 12, tablet: 3 }}>
    <Box csx={{ ...palette('teal'), padding: 2 }}>3 units</Box>
  </Column>
</Columns>
```
