## Usage

```jsx
import { Columns, Column } from '@vtex/admin-ui'

function Example() {
  return (
    <Columns>
      <Column>{content}</Column>
      <Column>{content}</Column>
      <Column>{content}</Column>
    </Columns>
  )
}
```

</Section>

<Section>

## Composition

| Name      | Description                                |
| --------- | ------------------------------------------ |
| `Columns` | A 12-column flexbox based columned layout. |
| `Column`  | As the name says, it implements a column   |

</Section>
