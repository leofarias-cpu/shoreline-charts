## Handling state

### Uncontrolled

```jsx live

```

### Controlled

```jsx live
function Example() {
  const [value, setValue] = useState('')

  return (
    <TextArea
      label="Controlled"
      value={value}
      onChange={(e) => setValue(e.target.value)}
    />
  )
}
```
