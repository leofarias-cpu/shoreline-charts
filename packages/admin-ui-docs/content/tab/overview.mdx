## What is this?

Tab triggers allow merchants to navigate between up to five views of content that are in the same level of hierarchy.

</IntroSection>

<Section>

## Example

```jsx live
function Example() {
  const state = useTabState()

  return (
    <>
      <TabList state={state}>
        <Tab>Tab 1</Tab>
        <Tab>Tab 2</Tab>
      </TabList>
      <TabPanel state={state}>Panel 1</TabPanel>
      <TabPanel state={state}>Panel 2</TabPanel>
    </>
  )
}
```

</Section>

## Manual navigation

```jsx live
function Example() {
  const state = useTabState()

  return (
    <>
      <TabList state={state}>
        <Tab id="1">Tab 1</Tab>
        <Tab id="2">Tab 2</Tab>
      </TabList>
      <TabPanel state={state} id="1">
        Panel 1
        <Button onClick={() => state.select('2')}>Navigate to tab 2</Button>
      </TabPanel>
      <TabPanel state={state} id="2">
        Panel 2
        <Button onClick={() => state.select('1')}>Navigate to tab 1</Button>
      </TabPanel>
    </>
  )
}
```
