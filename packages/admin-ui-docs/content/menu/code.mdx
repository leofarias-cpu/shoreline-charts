## Usage

```jsx
import {
  MenuButton,
  Menu,
  MenuItem,
  MenuDivider,
  useMenuState,
} from '@vtex/admin-ui'

function Example() {
  const state = useMenuState()

  return (
    <>
      <MenuButton state={state} />
      <Menu state={state}>
        <MenuItem label="Create" icon={<IconPlus />} />
        <MenuItem label="Edit" icon={<IconPencil />} />
        <MenuItem label="Download" icon={<IconArrowLineDown />} />
      </Menu>
    </>
  )
}
```

## Accessibility

- `Menu` has role `menu`.
- `MenuItem` has role `menuitem`.
- Pressing <kbd>Enter</kbd> on `MenuButton` opens the `Menu` and places focus on its first item.
- When `MenuItem` has focus, <kbd>Space</kbd> and <kbd>Enter</kbd> triggers it.

</Section>

### Best Practices

- You always must set the `aria-label` property in the `Menu` component.

```jsx
function Example() {
  const state = useMenuState()

  return (
    <>
      <MenuButton state={state} />
      <Menu aria-label="actions" state={state}>
        <MenuItem label="Create" icon={<IconPlus />} />
        <MenuItem label="Edit" icon={<IconPencil />} />
        <MenuItem label="Download" icon={<IconArrowLineDown />} />
        <MenuDivider />
        <MenuItem label="Delete" icon={<IconTrash />} critical />
      </Menu>
    </>
  )
}
```
