<IntroSection>

## What is this?

Card containers allow merchants to easily navigate in a page with dense or extensive content.

</IntroSection>

<br />

:::info
The following recommendations address the application of cards in designs, regardless of whether they are technically a Card or not.
:::

<br />

<Section>

## Container

```jsx live
<Card>
  <Center className={csx({ width: '100%', height: 250, bg: '$secondary' })}>
    Content
  </Center>
</Card>
```

</Section>

<Section>

## Title

```jsx live
<Card>
  <CardHeader>
    <CardTitle>Title</CardTitle>
  </CardHeader>
  <CardContent className={csx({ width: '100%', bg: '$secondary' })}>
    <Center className={csx({ height: 250 })}>Content</Center>
  </CardContent>
</Card>
```

</Section>
