## What is this?

TextInput containers allow merchants to input text that takes up to one line in a form.

<Section>

## Handling state

### Uncontrolled

```jsx live
<TextInput label="Controlled" />
```

### Controlled

```jsx live
function Example() {
  const [value, setValue] = useState('')

  return (
    <TextInput
      label="Controlled"
      value={value}
      onChange={(e) => setValue(e.target.value)}
    />
  )
}
```
