## What is this?

Dividers serve the purpose of separating and distinguishing sections of content or groups of items. The divider component renders an `hr` element and grants accessibility described on the [WAI-ARIA Separator Role](https://www.w3.org/TR/wai-aria-1.1/#separator).

</IntroSection>

<Section hasExplanation>

## Horizontal

By default, the orientation property has a `horizontal`.

```jsx live
<Card csx={{ width: 500 }}>
  <Box
    csx={{
      bg: '$info',
      height: '100px',
      width: '100%',
    }}
  />

  <Divider csx={{ marginY: 6 }} />

  <Box
    csx={{
      bg: '$info',
      height: '100px',
      width: '100%',
    }}
  />
</Card>
```
</Section>

<Section hasExplanation>

## Vertical

 The value can also be set to `vertical`

```jsx live
<Card csx={{ width: 600 }}>
  <Columns>
    <Column>
      <Box
    csx={{
      bg: '$info',
      height: '100px',
      width: '100%',
    }}
  />
    </Column>

    <Divider orientation="vertical" />

    <Column>
      <Box
    csx={{
      bg: '$info',
      height: '100px',
      width: '100%',
    }}
  />
    </Column>
  </Columns>
</Card>
```

</Section>
