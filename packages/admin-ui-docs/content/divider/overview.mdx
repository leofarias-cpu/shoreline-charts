## Horizontal

By default, the orientation property has a `horizontal`.

```jsx live
<div>
  <div
    className={csx({
      bg: '$info',
      height: '100px',
      width: '100%',
    })}
  />

  <Divider className={csx({ marginY: 6 })} />

  <div
    className={csx({
      bg: '$info',
      height: '100px',
      width: '100%',
    })}
  />
</div>
```

</Section>

<Section hasExplanation>

## Vertical

The value can also be set to `vertical`

```jsx live
<Flex>
    <div
      className={csx({
        bg: '$info',
        height: '100px',
        width: '1/2',
      })}
    />

  <Divider orientation="vertical" className={csx({ marginX: 6 })} />

    <div
      className={csx({
        bg: '$info',
        height: '100px',
        width: '1/2',
      })}
    />
</Flex>
```

</Section>
