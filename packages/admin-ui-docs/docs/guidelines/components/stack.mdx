Stack is a layout component that makes it easy to stack elements together and apply a space between them.

## Examples

### Space

The space between components can be controlled using the `space` property.

```jsx live
<Stack space="$xl">
  <Button>Button 1</Button>
  <Button>Button 2</Button>
  <Button>Button 3</Button>
  <Button>Button 4</Button>
  <Button>Button 5</Button>
</Stack>
```

### Direction

The direction can be either `row` or `column` (default). It can be controlled with the `direction` property.

```jsx live
<Stack direction="row">
  <Button>Button 1</Button>
  <Button>Button 2</Button>
  <Button>Button 3</Button>
  <Button>Button 4</Button>
  <Button>Button 5</Button>
</Stack>
```

### Fluid

While on `column` direction, you can alternatively use the `fluid` property. This will make the inner children, match the container width.

```jsx live
<Box csx={{ maxWidth: 320 }}>
  <Stack fluid>
    <Button>Button 1</Button>
    <Button>Button 2</Button>
    <Button>Button 3</Button>
    <Button>Button 4</Button>
    <Button>Button 5</Button>
  </Stack>
</Box>
```

### Align

You can set items alignment (`start` [default], `end`) through the `align` prop.

```jsx live
<Stack align="end">
  <Button>First</Button>
  <Button>Second</Button>
  <Button>Third</Button>
</Stack>
```

### Responsive

```jsx live
<Stack
  direction={{
    mobile: 'column',
    tablet: 'row',
  }}
  fluid
>
  <Button>First</Button>
  <Button>Second</Button>
  <Button>Third</Button>
  <Button>Fourth</Button>
  <Button>Fith</Button>
</Stack>
```

```jsx isStatic
import { Stack } from '@vtex/admin-ui'

function Example() {
  return <Stack>{/* Stacked elements */}</Stack>
}
```
