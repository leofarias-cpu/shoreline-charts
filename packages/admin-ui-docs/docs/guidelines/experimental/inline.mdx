Render a set of components in a row with equal spacing between them. It also supports wrapping.

## Usage

```jsx
import { experimental_Inline as Inline } from '@vtex/admin-ui'

function Example() {
  return (
    <Inline>
      <Tag label="Brazil" />
      <Tag label="Argentina" />
      <Tag label="Chile" />
      <Tag label="Uruguai" />
    </Inline>
  )
}
```

## Examples

### Basic

```jsx noInline live
const Inline = experimental_Inline

function Example() {
  return (
    <Inline>
      <Tag label="Brazil" />
      <Tag label="Argentina" />
      <Tag label="Chile" />
      <Tag label="Uruguai" />
    </Inline>
  )
}

render(<Example />)
```

### Wrap

Wrapping is applied automatically.

```jsx noInline live
const Inline = experimental_Inline

function Example() {
  return (
    <Box csx={{ width: '18rem' }}>
      <Inline>
        <Tag label="Brasil" />
        <Tag label="Argentina" />
        <Tag label="Chile" />
        <Tag label="Uruguai" />
        <Tag label="Colombia" />
        <Tag label="Equador" />
        <Tag label="Venezuela" />
      </Inline>
    </Box>
  )
}

render(<Example />)
```

### Space

Space is configured using the `vSpace` and `hSpace`, with either token or number values. In the case of a numeric value, the unit is `rem` by default. You can also set the `unit` prop to change it.

```jsx noInline live
const Inline = experimental_Inline

function Example() {
  return (
    <Box csx={{ width: '18rem' }}>
      <Inline vSpace="$xl" hSpace="$xl">
        <Tag label="Brasil" />
        <Tag label="Argentina" />
        <Tag label="Chile" />
        <Tag label="Uruguai" />
        <Tag label="Colombia" />
        <Tag label="Equador" />
        <Tag label="Venezuela" />
      </Inline>
    </Box>
  )
}

render(<Example />)
```
