# Bleed

Sometimes a component needs to trespass the parent padding to achieve alignment harmony and visual compensation to the layout. You can accomplish this with `Bleed`, which applies a negative margin to represent the amount of trespassing desired.

## Usage

```jsx
import { experimental_Bleed as Bleed } from '@vtex/admin-ui'

function Example() {
  return (
    <Card>
      <Bleed>
        <img src="..." />
      </Bleed>
    </Card>
  )
}
```

## Examples

### Basic

```jsx noInline live
const Bleed = experimental_Bleed

const containerWidth = '20rem'

function Example() {
  return (
    <Box
      csx={{
        padding: '1rem',
        border: '$neutral',
        width: containerWidth,
      }}
    >
      <Bleed top={1} left={1} right={1}>
        <tag.img
          csx={{
            width: containerWidth,
          }}
          src="https://vtex.com/wp-content/uploads/2020/04/VTEX-Brand.svg"
          alt="vtex-brand"
        />
      </Bleed>
      <Paragraph>
        VTEX is the Enterprise Digital Commerce Platform for brands and
        retailers looking to build a future-proof digital commerce strategy.
        Powered by a modern platform and a team of digital commerce experts, we
        enable enterprises to achieve faster time-to-market, reach their
        customers across any channel and uncover new areas of growth.
      </Paragraph>
    </Box>
  )
}

render(<Example />)
```

### With Inline

```jsx noInline live
const Bleed = experimental_Bleed
const Inline = experimental_Inline

function Example() {
  return (
    <Bleed top="$xl" left="$xl">
      <Inline vSpace="$xl" hSpace="$xl">
        <Tag label="Brazil" />
        <Tag label="Argentina" />
        <Tag label="Chile" />
        <Tag label="Uruguai" />
      </Inline>
    </Bleed>
  )
}

render(<Example />)
```
