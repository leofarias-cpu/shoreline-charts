# Icons

The icon set of the AdminUI.

## Usage

```jsx isStatic
import { IconName } from '@vtex/admin-ui'
```

```jsx
<Set>
  <IconCreditCard />
  <IconWallet />
  {/** With directions */}
  <IconArrowUp direction="up" /> {/** default */}
  <IconArrowUp direction="right" />
  <IconArrowUp direction="down" />
  <IconArrowUp direction="left" />
  <IconBell />
</Set>
```
