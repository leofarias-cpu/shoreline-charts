# Sizes

## Props within style object

`width`, `height`, `maxWidth`, `maxHeight`, `minWidth`, `minHeight`

## Multiples

| PROP   | COMBINES          | DESCRIPTION                                               |
| ------ | ----------------- | --------------------------------------------------------- |
| `size` | `width`, `height` | Element size, useful to create a square or perfect circle |

## Scales

### Fixed size

| TOKEN | REM     | PX      |
| ----- | ------- | ------- |
| sm    | `20rem` | `20rem` |
| md    | `48rem` | `48rem` |
| lg    | `56rem` | `56rem` |
| xl    | `64rem` | `64rem` |

### Full space

| TOKEN        | PERCENTAGE |
| ------------ | ---------- |
| full         | `100%`     |
| screenHeight | `100vh`    |
| screenWidth  | `100vw`    |
