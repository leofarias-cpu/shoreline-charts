### Composites

List is a compound component with the following composites:

| Name            | Description                                                      | Props            |
| --------------- | ---------------------------------------------------------------- | ---------------- |
| `List.Item`     | List item. Renders a `div` element                               | `List.ItemProps` |
| `List.TextGrop` | Combine tilte, subtilte and description. Renders a `div` element | `BoxProps`       |

**List.Item Props**

Same as `BoxProps`.

**List.TextGroup Props**

Same as `ButtonProps` with aditional,

### Styled

You can also customize the list style with the `styled` HOC.

```jsx
import { List, styled } from '@vtex/admin-ui'

const StyledList = styled(List)`<custom_css_here>`
const StyledItem = styled(List.Item)`<custom_css_here>`

function Example() {
  return (
    <StyledList>
      <StyledItem>{/** custom styles */}</StyledItem>
    </StyledList>
  )
}
```
