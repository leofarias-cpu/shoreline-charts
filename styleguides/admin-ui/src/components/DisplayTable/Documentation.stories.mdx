## Customization

You can use three sx properties:

| prop    | description                   |
| ------- | ----------------------------- |
| sx      | main style                    |
| leadSx  | style of each lead container  |
| fieldSx | style of each field container |

## Limitations

- No empty state yet
- No actions within rows
- No `:hover` styles
- No virtualization
- No dnd
- No nesting
